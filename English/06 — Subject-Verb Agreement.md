# 06 — Subject-Verb Agreement
[[📝 English — INDEX|← Back to Index]]

---

## Questions From Your Papers

| Sentence | Answer | Rule |
|----------|--------|------|
| I, as well as he, _____ to be blamed | **am** | Verb agrees with first subject (I) |
| We all _____ playthings in the hands of God | **are** | We = plural |
| The commander, with all his soldiers, _____ killed | **was** | Commander = singular (with = not part of subject) |
| None _____ him | **likes** | None = singular |

---

## Core Rule

```
Singular subject → singular verb (is/was/has/does/V+s)
Plural subject → plural verb (are/were/have/do/V)
```

---

## Tricky Cases (All From Your Papers)

### Rule 1 — "As well as / along with / together with / with"
The verb agrees with the **FIRST subject** only
```
I, as well as he, **am** to be blamed. (I = first = singular → am)
The commander, with all his soldiers, **was** killed. (commander = singular → was)
Ram, along with his friends, **is** going. (Ram = singular → is)
```

### Rule 2 — "Either/Neither" (singular)
```
Either of the boys **is** correct.
Neither of them **was** present.
Either he or she **is** wrong.
```

### Rule 3 — "Each / Every / None / Nobody / Anybody"
```
Each student **has** a book. (singular)
Everyone **is** happy. (singular)
None of them **likes** him. (singular)
Nobody **knows** the answer. (singular)
```

### Rule 4 — Collective Nouns
```
The team **is** playing well. (team acts as one unit → singular)
The committee **has** decided. (singular)
The crowd **was** cheering. (singular)
```
> Exception: If members of group act individually → plural
> "The jury **are** divided in their opinions."

### Rule 5 — "A number of" vs "The number of"
```
A number of students **are** absent. (plural)
The number of students **is** 50. (singular)
```

### Rule 6 — Indefinite Pronouns
```
Somebody, anybody, everyone, no one, each → singular verb
Some, many, few, both → plural verb
```

### Rule 7 — Titles / Books / Countries
```
"Gulliver's Travels" **is** a famous book. (one work → singular)
The United States **is** a powerful country. (one entity → singular)
```

### Rule 8 — Arithmetic
```
Two plus two **is** four. (singular)
Ten percent of the students **are** absent. (plural noun → plural)
```

---

## Quick Reference Card

| Subject | Verb |
|---------|------|
| I | am / was / have |
| He/She/It | is / was / has / does / V+s |
| You/We/They | are / were / have / do / V |
| None/Each/Every | **singular** |
| A number of + plural noun | **plural** |
| The number of | **singular** |
| "as well as" | agrees with FIRST subject |
| "with / along with" | agrees with FIRST subject |

## Exam Traps ⚠️
> "I, as well as he" → **am** (NOT are — verb agrees with I)
> "None likes him" → **likes** (singular — NOT like)
> "A number of" → **plural**; "The number of" → **singular**
> Collective nouns (team, committee) → usually **singular**
> "Either of" → **singular** verb always
