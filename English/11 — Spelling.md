|---------|---------------|

| **commitment** | committment (extra t) |
| **committee** | commitee / comittee |
| **occasion** | occassion (double s wrong) |
| **necessary** | neccessary / necesary |

### Common Difficult Words
| Correct Spelling | Remember |
|-----------------|---------|
| **privilege** | No d — privi-LEGE not privile-DGE |
| **leisure** | lei-sure (l-e-i) |
| **colleague** | col-LEAGUE |
| **separate** | sep-A-rate (there's a rat in separate) |
| **definitely** | definite + ly |
| **occurrence** | occur + rence (double c, double r) |
| **vacuum** | one c, double u |
| **exaggerate** | double g |
| **conscience** | con-SCIENCE |
| **refrigerator** | refri-GE-rator (no d) |
| **immediately** | immediate + ly |
| **especially** | especial + ly |
| **occasion** | one c, double s — oc-CA-sion |
| **accommodation** | two c's and two m's |
| **necessary** | one c, two s's — ne-C-e-SS-ary |
| **recommend** | one c, one m — re-COM-mend |

## Quick Memory Tricks
- **Accommodation** = has 2 Cots and 2 Mattresses (2c, 2m)
- **Necessary** = 1 Collar, 2 Socks (1c, 2s)
- **Separate** = there's A RAT in sepARAte
- **Believe** = never beLIEve a LIE
- **Privilege** = it's a privi-LEGE (not ledge)

## Exam Traps ⚠️
> Accommodation = **two c's AND two m's** — most people miss one
> Occasion = **one c, two s's** (oc-ca-sion) — often written with double c
> Environment = **environment** (the n after environ is missed)
> Privilege = **no d** — not "priviledge"
> Leisure = **lei** not lie (l-e-i, not l-i-e)
