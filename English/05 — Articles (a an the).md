# 05 — Articles (a an the)
[[📝 English — INDEX|← Back to Index]]

---

## Questions From Your Papers

| Sentence | Answer | Rule |
|----------|--------|------|
| _____ dog is a faithful animal | **The** | General statement about a species |
| The leader gave me _____ | **a piece of work** | Countable, first mention |

---

## The 3 Rules — Simple Version

| Article | Use When |
|---------|---------|
| **a** | Before consonant **sounds** (first mention, singular) |
| **an** | Before vowel **sounds** (first mention, singular) |
| **the** | Specific/known noun, only one exists, mentioned before |

---

## A vs AN — Sound, Not Spelling

| Use A | Use AN |
|-------|--------|
| a **u**niversity (sounds like "yu") | an **u**mbrella (sounds like "u") |
| a **e**uropean (sounds like "yu") | an **e**agle |
| a **h**ouse | an **h**our (h is silent — sounds like "our") |
| a **o**ne-way street (sounds like "wu") | an **o**range |
| a **u**seful tool (sounds like "yu") | an **u**ncle |

---

## When to Use THE

| Rule | Example |
|------|---------|
| Already mentioned before | I saw **a** dog. **The** dog was big. |
| Only one exists | **The** sun, **the** moon, **the** sky |
| Superlatives | **The** best, **the** tallest, **the** first |
| Oceans, rivers, mountain ranges | **The** Amazon, **the** Himalayas |
| Countries with plural/common nouns | **The** USA, **the** UK, **the** Netherlands |
| Specific groups | **The** poor, **the** rich, **the** blind |
| Newspapers | **The** Times of India, **The** Hindu |
| Musical instruments | She plays **the** piano |
| General class (species) | **The** dog is a faithful animal |

---

## When NOT to Use Any Article

| Rule | Example |
|------|---------|
| Plural nouns in general | **Dogs** are faithful (NOT The dogs) |
| Proper nouns (most) | **India**, **Aizawl**, **Nehru** |
| Languages | She speaks **Hindi** |
| Meals | Have **breakfast**, eat **lunch** |
| Sports | He plays **football** |
| School/college in general purpose | He goes to **school** (as a student) |

---

## Exam Traps ⚠️
> **An** hour — NOT a hour (h is silent)
> **A** university — NOT an university (sounds like "yu")
> **A** European — NOT an European
> **The** dog is faithful = speaking of dogs as a species — correct
> No article before meals: have **breakfast** NOT have **a breakfast**
> **The** Himalayas (mountain range), **The** Amazon (river) — always use the
