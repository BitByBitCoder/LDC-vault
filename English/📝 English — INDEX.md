## ⚡ Flash Rules — Memorize These First

| Rule | Example |
|------|---------|
| Use **a** before consonant sounds | a dog, a university |
| Use **an** before vowel sounds | an apple, an hour |
| Use **the** for specific/known nouns | The dog is loyal |
| **is/am/are** = present; **was/were** = past | She **is** here; He **was** here |
| Passive = object + be + past participle | The house **was painted** by them |
| Indirect speech — present → past | "I am" → he said he **was** |
| **Since** = point of time; **For** = duration | since 2020; for 5 years |
| **Between** = two; **Among** = more than two | between you and me; among all of us |
| Subject-verb: singular subject → singular verb | The team **is** playing |
| Avoid double negatives | NOT "He doesn't know nothing" |

## 🚨 Common Exam Traps (From Your Papers)

| Wrong | Correct | Rule |
|-------|---------|------|
| She **doesn't likes** coffee | She **doesn't like** coffee | After does/doesn't — base verb |
| She **don't** like coffee | She **doesn't** like coffee | She = singular → doesn't |
| **Hardly had finished** he | **Hardly had he finished** | Inversion after negative adverbs |
| The tailor **is stitching** (passive → active) | The tailor **was stitching** | Tense must match |
| None **like** him | None **likes** him | None = singular |
| **Accommodate** | accomodation / accomoddate | Double c, double m |
| **Environment** | enviroment / environmant | No extra letters |
| **Privilege** | priviledge / privelege | No d in privilege |
| **Leisure** | lezure / liesure | ei not ie |
| Famous **for** | famous with / famous in | Famous for = correct preposition |
| He lives **on** Peter's Street | in / at | Live **on** a street |
