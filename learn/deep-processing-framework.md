# 🧠 Deep Processing Framework

## Step 0 — Identify note type first
| Type | Example | Method Focus |
|---|---|---|
| **Concept note** | Preamble, Directive Principles | Understanding — explain *why* |
| **Fact table** | Geography, dates, lists | Discrimination — tell confusables apart |

---

## For CONCEPT notes

1. **Close-book recall** — read once, close, reconstruct from memory. Check gaps.
2. **Feynman check** — explain it to an imaginary 12-year-old, no jargon.
3. **Answer Thinking Questions cold** — before checking, commit to an answer.
4. **Self-write one trap MCQ** — plausible wrong answer included.
5. **Spaced recall** — retest same note in 3 days, not today.

---

## For FACT-TABLE notes

1. **Cover the answer column** — go down questions only, say answer before checking.
2. **Contrast confusables in pairs** — e.g. Kanchenjunga vs Everest, Anaimudi vs Nilgiri. Ask: *why is X right and not Y?*
3. **Build one spatial/causal map** — merge separate tables into one journey/timeline/flow instead of isolated lists.
4. **Predict the trap before reading Exam Traps section** — "If I wrote a tricky MCQ here, what wrong option would I use?"
5. **Narrate from memory** — close the note, recite the whole map/journey out loud.

---

## Universal deep-processing filter (apply to both types)

Before checking a topic off, ask:

> **"Could I write an original exam question on this, with a plausible wrong answer?"**

- Yes → understood structurally, move on.
- No → still shallow, review again before moving on.

---

## Daily loop (short time window)

```
Read once
   ↓
Close-book recall
   ↓
2-3 hr gap → recall again (no notes)
   ↓
Self-write 1 trap question per topic
   ↓
Before sleep → final blank recall pass
```

## Obsidian callout shortcut

```
> [!question]- Prompt / question here
> Answer here (hidden until expanded)
```
- `-` = collapsed by default → forces recall on open
- `+` = expanded by default → use only for brand-new topics

**Rule:** Answer before you click to expand. Opening it fast = rereading with extra clicks.
