Just like Excel and Word, PowerPoint has its own set of essential shortcuts. While it shares many of the same text formatting shortcuts as Word, PowerPoint’s real power lies in shortcuts for manipulating objects (shapes, images) and running the slideshow itself.

Here are the most essential keyboard shortcuts to speed up your presentation building and delivery.

### **General & File Management**

The basics for managing your presentation files and adding slides.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + N**|Create a new presentation|
|**Ctrl + O**|Open an existing presentation|
|**Ctrl + S**|Save the active presentation|
|**Ctrl + M**|Insert a new slide|
|**Ctrl + D**|Duplicate the selected slide (when in the thumbnail pane)|
|**Ctrl + Z / Y**|Undo / Redo the last action|
|**Ctrl + P**|Print the presentation|

### **Slideshow Delivery**

These are critical for when you are actually presenting to an audience.

|**Shortcut**|**Action**|
|---|---|
|**F5**|Start the slideshow from the first slide|
|**Shift + F5**|Start the slideshow from the current slide|
|**Esc**|End the slideshow|
|**N** or **Spacebar**|Advance to the next slide or animation|
|**P** or **Backspace**|Return to the previous slide or animation|
|**Number + Enter**|Jump to a specific slide number (e.g., typing `5` then `Enter` goes to slide 5)|
|**B**|Pause the show and display a black screen (press `B` again to resume)|
|**W**|Pause the show and display a white screen (press `W` again to resume)|
|**Ctrl + P**|Change the pointer to a pen to draw on the slide|
|**Ctrl + L**|Change the pointer to a laser pointer|
|**E**|Erase all on-screen drawings|

### **Working with Objects (Shapes, Images, Text Boxes)**

PowerPoint is highly visual, so manipulating objects quickly is a massive time-saver.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + D**|Duplicate the selected object(s) (faster than Copy/Paste)|
|**Ctrl + G**|Group selected objects together|
|**Ctrl + Shift + G**|Ungroup selected objects|
|**Arrow Keys**|Nudge the selected object slightly|
|**Ctrl + Arrow Keys**|Micro-nudge the selected object for precise alignment|
|**Shift + Arrow Keys**|Resize the selected object|
|**Shift + Click/Drag**|Keep proportions constrained while drawing or resizing an object|
|**Alt + Left / Right Arrow**|Rotate the selected object by 15 degrees|

### **Navigation & Selection**

Moving around your slides and selecting elements within them.

|**Shortcut**|**Action**|
|---|---|
|**Page Down / Page Up**|Move to the next / previous slide|
|**Tab**|Cycle forward through objects/placeholders on a slide|
|**Shift + Tab**|Cycle backward through objects/placeholders on a slide|
|**Ctrl + A**|Select all objects on the current slide (or all text if inside a text box)|
|**Alt + F5**|Open Presenter View (if you have two screens)|

### **Text Formatting**

These should feel very familiar if you use Microsoft Word.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + B / I / U**|Apply or remove Bold, Italic, or Underline|
|**Ctrl + Shift + > / <**|Increase or decrease font size|
|**Ctrl + L / E / R / J**|Align text Left / Center / Right / Justify|
|**Shift + F3**|Toggle text case (lowercase, UPPERCASE, Title Case)|
|**Ctrl + K**|Insert a hyperlink|
