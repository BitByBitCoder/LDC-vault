Just like Excel, Microsoft Word has an massive library of shortcuts. I have categorized the most powerful and frequently used ones so you can navigate, edit, and format documents without breaking your typing flow.

### **General & File Management**

The foundational shortcuts for managing your document and basic tasks.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + N**|Create a new document|
|**Ctrl + O**|Open an existing document|
|**Ctrl + S**|Save the active document|
|**F12**|Open the "Save As" dialog box|
|**Ctrl + P**|Print the document|
|**Ctrl + Z**|Undo the last action|
|**Ctrl + Y**|Redo the last action|
|**Ctrl + F**|Open the Navigation/Find pane|
|**Ctrl + H**|Open the Find and Replace dialog|
|**Ctrl + Enter**|Insert a page break|

### **Navigation**

Move your cursor exactly where you need it in a fraction of a second.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + Left / Right Arrow**|Move the cursor one word to the left/right|
|**Ctrl + Up / Down Arrow**|Move the cursor one paragraph up/down|
|**Home**|Move to the beginning of the current line|
|**End**|Move to the end of the current line|
|**Ctrl + Home**|Move to the beginning of the document|
|**Ctrl + End**|Move to the end of the document|
|**Page Up / Page Down**|Move up or down one screen|
|**Alt + Ctrl + Page Up / Down**|Move to the top or bottom of the visible window|

### **Selecting Text**

Highlight text quickly without dragging a mouse.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + A**|Select the entire document|
|**Shift + Arrow Keys**|Select text one character at a time|
|**Ctrl + Shift + Arrow Keys**|Select text one word at a time|
|**Shift + Home**|Select text from the cursor to the beginning of the line|
|**Shift + End**|Select text from the cursor to the end of the line|
|**F8**|Turn on "Extend Selection" mode (press again to expand selection)|

### **Editing & Deleting**

Manipulate words and lines faster.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + X / C / V**|Cut / Copy / Paste|
|**Ctrl + Backspace**|Delete one word to the left of the cursor|
|**Ctrl + Delete**|Delete one word to the right of the cursor|
|**Shift + Enter**|Insert a soft line break (new line without a new paragraph)|
|**Ctrl + K**|Insert a hyperlink|
|**Shift + F7**|Open the Thesaurus for the selected word|

### **Character Formatting**

Change how your text looks directly from the keyboard.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + B / I / U**|Apply or remove Bold, Italic, or Underline|
|**Ctrl + Shift + > / <**|Increase or decrease font size by one point|
|**Ctrl + ] / [**|Increase or decrease font size exactly 1 point|
|**Shift + F3**|Toggle text case (lowercase, UPPERCASE, Title Case)|
|**Ctrl + =** (Equals)|Apply subscript formatting (e.g., H₂O)|
|**Ctrl + Shift + +** (Plus)|Apply superscript formatting (e.g., x²)|
|**Ctrl + Spacebar**|Remove all manual character formatting|

### **Paragraph Formatting & Alignment**

Structure your lines and blocks of text perfectly.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + L / E / R / J**|Align text Left / Center / Right / Justify|
|**Ctrl + 1**|Single-space lines|
|**Ctrl + 2**|Double-space lines|
|**Ctrl + 5**|1.5-line spacing|
|**Ctrl + M**|Indent the paragraph|
|**Ctrl + Shift + M**|Remove paragraph indent|
|**Ctrl + T**|Create a hanging indent|
|**Ctrl + Q**|Remove all manual paragraph formatting|

### **Styles & Structure**

For managing long reports, essays, or structured documents.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + Alt + 1**|Apply Heading 1 style|
|**Ctrl + Alt + 2**|Apply Heading 2 style|
|**Ctrl + Alt + 3**|Apply Heading 3 style|
|**Ctrl + Shift + N**|Apply the Normal style|
|**Alt + Shift + Up / Down**|Move the selected paragraph up or down in the document|
