While there are hundreds of keyboard shortcuts in Microsoft Excel, listing every single one can be overwhelming. To make this practical and highly scannable, I have organized the most essential and powerful shortcuts by category.

Mastering these will significantly speed up your workflow.

### **General & File Management**

These are the foundational shortcuts for managing your workbook and general tasks.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + N**|Create a new workbook|
|**Ctrl + O**|Open an existing workbook|
|**Ctrl + S**|Save the active workbook|
|**F12**|Open the "Save As" dialog box|
|**Ctrl + P**|Print the workbook|
|**Ctrl + Z**|Undo the last action|
|**Ctrl + Y**|Redo the last action|
|**Ctrl + F**|Open the Find dialog|
|**Ctrl + H**|Open the Find and Replace dialog|

### **Navigation**

Use these to move around large spreadsheets quickly without touching your mouse.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + Arrow Keys**|Move to the edge of the current data region|
|**Ctrl + Home**|Move to the beginning of the worksheet (Cell A1)|
|**Ctrl + End**|Move to the last cell on a worksheet containing data|
|**Page Down / Page Up**|Move one screen down / one screen up|
|**Alt + Page Down / Up**|Move one screen to the right / to the left|
|**Ctrl + Page Down / Up**|Switch to the next / previous worksheet tab|
|**F5** or **Ctrl + G**|Open the "Go To" dialog box|

### **Selecting Data**

Selecting rows, columns, or massive ranges is much faster with the keyboard.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + A**|Select the entire worksheet (or current data region)|
|**Shift + Space**|Select the entire row|
|**Ctrl + Space**|Select the entire column|
|**Ctrl + Shift + Arrow**|Extend the selection to the edge of the data region|
|**Shift + Arrow Keys**|Extend the selection by one cell|
|**Ctrl + Shift + Home**|Extend selection to the beginning of the worksheet|

### **Editing & Data Entry**

Shortcuts to help you enter and manipulate data inside the cells.

|**Shortcut**|**Action**|
|---|---|
|**F2**|Edit the active cell and put the cursor at the end|
|**Alt + Enter**|Start a new line (line break) within the same cell|
|**Ctrl + D**|Fill down (copies the content/format from the cell above)|
|**Ctrl + R**|Fill right (copies the content/format from the cell to the left)|
|**Ctrl + ;** (Semicolon)|Enter the current date|
|**Ctrl + Shift + :** (Colon)|Enter the current time|
|**Ctrl + E**|Flash Fill (automatically recognizes patterns and fills data)|

### **Formulas & Functions**

Crucial shortcuts for building and auditing formulas.

|**Shortcut**|**Action**|
|---|---|
|**Alt + =** (Equals)|AutoSum (automatically inserts the SUM function)|
|**F4**|Toggle absolute/relative cell references (e.g., A1 to $A$1)|
|**Shift + F3**|Open the Insert Function dialog box|
|**Ctrl + `** (Grave Accent)|Toggle between displaying cell values and formulas|
|**F9**|Calculate all worksheets in all open workbooks|

### **Formatting**

Quickly change how your data looks without hunting through the ribbon menu.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + 1**|Open the Format Cells dialog box|
|**Ctrl + B / I / U**|Apply or remove Bold, Italic, or Underline|
|**Ctrl + Shift + $**|Apply Currency format with two decimal places|
|**Ctrl + Shift + %**|Apply Percentage format with no decimal places|
|**Ctrl + Shift + #**|Apply Date format (Day, Month, Year)|
|**Ctrl + Shift + !**|Apply Number format with two decimal places|
|**Ctrl + Shift + &**|Apply an outline border to the selected cells|
|**Ctrl + Shift + _**|Remove the outline border from the selected cells|

### **Managing Rows & Columns**

Shortcuts for modifying the structure of your grid.

|**Shortcut**|**Action**|
|---|---|
|**Ctrl + +** (Plus)|Insert blank cells, rows, or columns|
|**Ctrl + -** (Minus)|Delete selected cells, rows, or columns|
|**Ctrl + 9**|Hide the selected rows|
|**Ctrl + Shift + 9**|Unhide any hidden rows within the selection|
|**Ctrl + 0** (Zero)|Hide the selected columns|
|**Ctrl + Shift + 0**|Unhide any hidden columns within the selection|
