## PowerPoint Interface

| Component | Description |
|-----------|-------------|
| **Slide** | Individual page in presentation |
| **Slide pane** | Main editing area |
| **Slide panel (left)** | Thumbnails of all slides |
| **Notes pane** | Add speaker notes below slide |
| **Placeholder** | Dotted box — holds text, images, charts |

## Slide Operations

| Action | How |
|--------|-----|
| Add new slide | **Ctrl+M** or Home → New Slide |
| Delete slide | **Select + Delete key** |
| Duplicate slide | Right-click → Duplicate |
| Move slide | Drag in Slide Sorter view |
| Hide slide | Right-click → Hide Slide |
| **Permanently remove** | **Select slide and press Delete** |

> ⚠️ Hide slide ≠ delete — hidden slides stay in file but don't show in presentation

## Exam Traps ⚠️
> Slide layout = **arrangement of placeholders** — NOT the theme or design colors
> Slide Sorter = **thumbnails + rearranging** — NOT editing
> Rehearse Timings = **records time per slide** — NOT applies slide layouts
> Group = **move as one object** — NOT align or distribute
> Delete slide permanently = **select + Delete key** — Hide = still in file but hidden
> Transition = between **slides** — Animation = objects **within** a slide
> Handouts = **multiple slides per page** for audience — not just one slide
