## PowerPoint Views

| View | Purpose |
|------|---------|
| **Normal View** | Default editing view — slide + notes panel |
| **Slide Sorter View** | **Rearranging and organizing all slides** |
| **Slide Show View** | **Full-screen presentation mode** |
| **Notes Page View** | Shows slide + speaker notes |
| **Reading View** | Full screen without editing tools |
| **Outline View** | Shows only text content of slides |

## Slide Basics

| Question | Answer |
|----------|--------|
| Add new slide | Home → New Slide / Ctrl + M |
| Duplicate slide | Right-click → Duplicate |

## Shortcuts — PowerPoint

|---------|--------|

## Exam Traps ⚠️
> Slide sorter = **rearranging slides** — NOT editing content
> Slide Show = **full-screen presentation** — NOT editing
> Slide layout = **arrangement of placeholders** — NOT overall design
> Rehearse Timings = **records time on each slide** — NOT animation timing
> Group = **move multiple objects as one** — NOT lock objects
> Delete slide permanently = **Select → Delete** — NOT hide (hide just hides it)
> Design Theme = overall look — Slide Layout = placeholder arrangement
> Normal view = **editing** — Slide Show = **presenting**
