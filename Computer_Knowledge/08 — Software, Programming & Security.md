## Software Types

| Type | Description | Examples |
|------|-------------|---------|
| **System Software** | Controls hardware — foundation | OS (Windows, Linux), Device drivers, BIOS |
| **Application Software** | Does specific user tasks | MS Office, Browser, Games, **Spreadsheet** |
| **Utility Software** | Maintains and optimizes | Antivirus, Backup, Disk cleaner |
| **Middleware** | Links different software | Database connectors |

> ⚠️ Spreadsheet = **Application software** — OS = **System software**

## Programming Concepts

| Term | Meaning |
|------|---------|
| **Algorithm** | Step-by-step instructions to solve a problem |
| **Flowchart** | Visual diagram of algorithm |
| **Pseudocode** | Algorithm written in plain English (not actual code) |
| **Compiler** | Translates entire high-level code to machine code at once |
| **Interpreter** | Translates code line by line |
| **Assembler** | Converts assembly language to machine code |
| **High-level language** | Human-readable (Python, Java, C++) |
| **Low-level language** | Machine/assembly language |
| **Machine language** | Binary (0s and 1s) — directly understood by CPU |
| **Source code** | Original human-written code |
| **Object code** | Machine code after compilation |
| **Bug** | Error in program |
| **Debugging** | Finding and fixing bugs |

---

## Compiler vs Interpreter

| Feature | Compiler | Interpreter |
|---------|----------|-------------|
| Translation | Whole program at once | Line by line |
| Speed | Faster execution | Slower execution |
| Error detection | After full compilation | At each line |
| Examples | C, C++, Java (compiler) | Python, JavaScript (interpreter) |

## Cyber Security

| Term | Meaning |
|------|---------|
| **Virus** | Self-replicating malicious program |
| **Worm** | Spreads over network without host file |
| **Trojan Horse** | Malware disguised as legitimate software |
| **Ransomware** | Encrypts files and demands payment |
| **Spyware** | Secretly monitors user activity |
| **Adware** | Shows unwanted advertisements |
| **Malware** | Any malicious software (umbrella term) |
| **Phishing** | Fake emails/sites to steal information |
| **Hacking** | Unauthorized access to systems |
| **Firewall** | Blocks unauthorized network access |
| **Antivirus** | Detects and removes malware |
| **Encryption** | Converts data to unreadable form |
| **Backup** | Copy of data for recovery |

## Exam Traps ⚠️
> Spreadsheet = **Application** software — NOT system software
> Compiler = **whole program** at once — Interpreter = **line by line**
> Virus = **self-replicating** — Worm = spreads on network without host
> Trojan = **disguised as legitimate** — doesn't self-replicate
> Ransomware = **encrypts AND demands payment** — specific type of malware
> Malware = **umbrella term** for all malicious software
> Primary key = **unique identifier** — every table must have one
> SQL = **Structured Query Language** — used for databases (NOT a programming language like C)
