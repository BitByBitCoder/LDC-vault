## Types of Software

| Type | Description | Examples |
|------|-------------|---------|
| **System Software** | Controls hardware — runs the computer | OS, Device Drivers, BIOS |
| **Application Software** | For specific user tasks | MS Word, Excel, Browser |
| **Utility Software** | Maintenance and optimization | Antivirus, Disk Cleaner |

### Application Software Examples
- Word Processor (MS Word)
- **Spreadsheet (MS Excel)** ← appeared in your papers
- Database (MS Access)
- Web Browser (Chrome)
- Media Player
- Games

### System Software Examples
- **Operating System** (Windows, Linux)
- **Device Drivers** (printer driver)
- **BIOS/UEFI** (firmware)
- **Compiler, Assembler, Interpreter**

## Language Translators

| Translator | What it does |
|-----------|-------------|
| **Compiler** | Translates **entire program** at once |
| **Interpreter** | Translates **line by line** |
| **Assembler** | Converts Assembly language to machine code |
| **Linker** | Links compiled modules together |
| **Loader** | Loads program into memory for execution |

### Compiler vs Interpreter
| Compiler | Interpreter |
|---------|-------------|
| Translates whole program first | Translates line by line |
| Faster execution | Slower execution |
| C, C++, Java | Python, BASIC |
| Errors shown after full compilation | Stops at first error |

## Hardware-Software Relationship

| Question | Answer |
|----------|--------|
| Hardware without software | Cannot function |
| Software instructs hardware | **To perform specific tasks** |
| Hardware interprets user commands | **Through software** |
| Software replaces hardware | **False** |

## Exam Traps ⚠️
> Application software = **spreadsheet, word processor** — NOT operating system
> OS = **system software** — NOT application software
> COBOL = **commercial** — FORTRAN = scientific
> Compiler = **whole program at once** — Interpreter = line by line
> Oracle = **database** — NOT antivirus
> Hardware + Software together = **computer system** — hardware alone cannot function
> Firmware = **embedded in hardware** — NOT installed by user
