## 🚨 Master Traps — These Appear Every Paper

| Trap | Wrong | Correct |
|------|-------|---------|
| Processing unit | Input unit / Storage | **CPU** |
| Cache between | Hard disk & monitor | **CPU and Main Memory** |
| Sequential access device | Hard disk / SSD | **Magnetic Tape** |
| Application software example | Operating System | **Spreadsheet / Word processor** |
| OS provides file management through | Task Manager | **File Explorer** |
| Shortcut contains | Duplicate file | **Link to original** |
| Log off | Shuts down computer | **Ends session only — computer stays on** |
| WordPad vs Notepad | Same thing | **WordPad supports formatting — Notepad does not** |
| Excel formula starts with | + or # | **= (equals)** |
| Relative reference | Stays fixed | **Changes when copied** |
| Absolute reference | Changes when copied | **Stays fixed ($A$1)** |
| Slide sorter | Edit slides | **Rearrange/organize slides** |
| WWW = Internet | Same thing | **WWW is a SERVICE on top of Internet** |
| Server stores | Requests services | **Stores and manages shared resources** |
| Digitizer | Scans text | **Converts graphical drawings to digital** |
| Wrap text | Merges cells | **Displays long text in multiple lines in a cell** |
| Merge & Center | Wraps text | **Combines cells and centers content** |
| COUNT() | Counts text | **Counts NUMERIC entries only** |
| COUNTA() | Counts numbers | **Counts ALL non-empty cells** |
