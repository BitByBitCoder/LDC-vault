## Computer Generations — OS Types

| Generation | Period | Technology | OS Type |
|-----------|--------|-----------|---------|
| 1st | 1940s–50s | Vacuum Tubes | No OS — batch processing |
| 2nd | 1950s–60s | Transistors | Batch OS |
| **3rd** | **1960s–70s** | **ICs** | **Time-sharing, real-time, networks, distributed** |
| 4th | 1970s–now | Microprocessors | GUI, multitasking |
| 5th | Future | AI/VLSI | AI-based |

## Exam Traps ⚠️
> Minimize/Maximize = **Title Bar** — NOT Taskbar or Scroll Bar
> WordPad = **.rtf** — Notepad = **.txt** — Word = **.docx**
> Different First Page = **removes page 1 number** — NOT Delete or Page Break
> Bullets & Numbering = **Home Tab** — NOT Insert Tab
> Hide row = **right-click row NUMBER** — NOT right-click worksheet tab
> S/MIME = **email encryption** — HTTPS = web encryption — different
> Incognito = hides **locally** — ISP and websites can still track you
> Digital divide = **ICT access gap** — NOT computer brand difference
> 3rd gen OS = **time-sharing and networks** — NOT just multitasking (that's 4th gen)
