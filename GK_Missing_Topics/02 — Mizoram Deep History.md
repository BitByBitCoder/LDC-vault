## Administrative History — Chronological

| Year | Event |
|------|-------|
| 1898 | Lushai Hills District formed under Assam |
| 1898 | North Lushai Hills + South Lushai Hills merged |
| 1935 | Lushai Hills declared "Excluded Area" |
| 1954 | Lushai Hills renamed **Mizo Hills District** |
| **15 April 1954** | Chieftainship abolished — **259 chiefs** stepped down |
| 1959 | Mautam Famine (bamboo flowering) |
| 1960 | Mizo Cultural Society formed |
| 1961 | MNF formed |
| **1 March 1966** | Mizo Buai (insurgency) started |
| **21 January 1972** | Union Territory — divided into **3 districts:** Aizawl, Chhimtuipui, Lunglei |
| 1972 | Mizo Hills District Council abolished |
| 1986 | Peace Accord signed |
| **20 February 1987** | Statehood — 23rd state of India |
| **3 June 2019** | 3 new districts added — Hnahthial, Khawzawl, Saitual |

> 💡 **After 1972, original 3 districts = Aizawl, Chhimtuipui, Lunglei** — Chhimtuipui is now part of Lawngtlai area
