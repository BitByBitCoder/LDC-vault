## Step-by-Step Method

```
Step 1: Convert all statements to > or < symbols
Step 2: Arrange in a single chain from biggest to smallest
Step 3: Answer the question from your chain
```

## Worked Examples From Your Papers

### Q1: A taller than B but shorter than C. D taller than A but shorter than C.
```
A > B (A taller than B)
C > A (A shorter than C)
D > A (D taller than A)
C > D (D shorter than C)

Chain: C > D > A > B
Tallest = C ✓
```

### Q2: A taller than B, shorter than C. D shorter than B.
```
C > A > B > D
Shortest = D ✓
```

### Q3: Muana>Siama, Muana<Rama, Rama>Rina, Rama<Kunga, Siama>Rina
```
Kunga > Rama > Muana > Siama > Rina
Second shortest = Siama

Wait — check question again: "second shortest"
Order shortest to tallest: Rina < Siama < Muana < Rama < Kunga
2nd shortest = Siama ✓
```

### Q4: Age problem — Awmpuia as much younger to Vana as he is older to Tanpuia
```
Vana − Awmpuia = Awmpuia − Tanpuia
→ Vana + Tanpuia = 2 × Awmpuia
→ 48 = 2 × Awmpuia
→ Awmpuia = 24 ✓
```

## Common Comparison Patterns

### Pattern 1 — Height/Age/Weight Chain
```
Always convert: "A is taller than B" → A > B
"A is shorter than B" → A < B or B > A
"A is as tall as B" → A = B

Then arrange in one chain
```

### Pattern 2 — Middle Person Problems
```
"Awmpuia as much younger to Vana as older to Tanpuia"
This means Awmpuia is exactly in the middle
Awmpuia's age = (Vana + Tanpuia) / 2
```

### Pattern 3 — Average Age = Middle Age
```
If X is equidistant from Y and Z:
X = (Y + Z) / 2
Y + Z = 2X
```

## Building the Chain — Tips

| Statement | Symbol |
|-----------|--------|
| A is taller/older/heavier/bigger than B | A > B |
| A is shorter/younger/lighter/smaller than B | B > A |
| A is as tall as B | A = B |
| A is second tallest | A is 2nd in the chain from top |
| A is second shortest | A is 2nd in the chain from bottom |

---

## Exam Traps ⚠️
> "Shorter than" = the other person is TALLER — rewrite as B > A
> Always build the **full chain** before answering — partial chain leads to wrong answer
> "Second shortest" = 2nd from bottom of chain — NOT 2nd from top
> "As much younger as older" = exactly in the middle → use average formula
> Multiple conditions — some might be redundant (already implied by other conditions)
> When chain has 5 people — write all 5 in order before answering
