## The 4 Rules — Memorize These

### Rule 1 — ALL + ALL = ALL
```
All A are B + All B are C → All A are C ✓
All A are B + All B are C → All C are A ✗ (NOT reversible)
```

### Rule 2 — ALL + NO = NO
```
All A are B + No B is C → No A is C ✓
All roses are flowers + No flower is blue → No rose is blue ✓
```

### Rule 3 — SOME + ALL = SOME
```
Some A are B + All B are C → Some A are C ✓
Some teachers are strict + All strict are disciplined → Some teachers are disciplined ✓
```

### Rule 4 — SOME + NO = SOME NOT
```
Some A are B + No B is C → Some A are NOT C ✓
```

---

## Conversion Rules (Important)

| Statement | Can be Converted To |
|-----------|-------------------|
| All A are B | Some B are A ✓ (but NOT All B are A) |
| No A is B | No B is A ✓ (fully reversible) |
| Some A are B | Some B are A ✓ (fully reversible) |
| Some A are not B | **Cannot be converted** |

## Worked Examples From Your Papers

### Q1: All roses are flowers. Some flowers are red.
```
Conclusion I: All roses are red → WRONG
(Some flowers are red ≠ all flowers are red, so can't conclude all roses are red)

Conclusion II: Some roses are red → WRONG
(We don't know WHICH flowers are red — might not include roses)

Conclusion III: Some flowers are roses → CORRECT
(Convert: All roses are flowers → Some flowers are roses ✓)

Answer: Only III follows
```

### Q2: All doctors are educated. Some educated are unemployed.
```
SOME + SOME = No definite conclusion
Some educated are unemployed — this SOME might or might not include doctors
Conclusion I: Some doctors are unemployed → WRONG (not certain)
Conclusion II: All unemployed are doctors → WRONG (definitely not)

Answer: Neither follows
```

### Q3: All fruits are healthy. No healthy thing is junk food.
```
ALL + NO = NO
All fruits are healthy + No healthy is junk food → No fruit is junk food ✓

Conclusion I: No fruit is junk food → CORRECT ✓
Conclusion II: Some junk foods are healthy → WRONG
(No healthy is junk food → No junk food is healthy — converted)

Answer: Only I follows
```

### Q4: Some teachers are strict. All strict are disciplined.
```
SOME + ALL = SOME
Some teachers are strict + All strict are disciplined → Some teachers are disciplined ✓

Conclusion I: Some teachers are disciplined → CORRECT ✓
Conclusion II: All disciplined are teachers → WRONG
(We can't reverse ALL — not all disciplined are teachers)

Answer: Only I follows
```

## Quick Decision Table

| Combination | Result |
|-------------|--------|
| All + All | All (conclusion possible) |
| All + No | No (conclusion possible) |
| All + Some | Some (conclusion possible) |
| Some + All | Some (conclusion possible) |
| Some + No | Some Not (conclusion possible) |
| Some + Some | **No definite conclusion** |
| No + No | **No definite conclusion** |
| No + All | Some Not (conclusion possible) |

---

## Exam Traps ⚠️
> "All A are B" does NOT mean "All B are A" — **not reversible**
> "Some A are B" DOES mean "Some B are A" — **reversible**
> "No A is B" DOES mean "No B is A" — **reversible**
> Some + Some = **no conclusion** — most common trap
> "Probably" type conclusions — treat as wrong in standard LDC syllogism
> Conclusion must **necessarily** follow — if only possible, it does NOT follow
