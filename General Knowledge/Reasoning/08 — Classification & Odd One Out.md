## Types of Odd One Out

### Type 1 — Biology/Science Category
```
Ask: What category do 3 belong to? Which one breaks that category?

Cat, Dog, Cow = Mammals
Snake = Reptile → Snake is odd one out

Amoeba, Hydra, Bacteria = Asexual reproduction
Human = Sexual reproduction → Human is odd one out
```

### Type 2 — Usage/Function Category
```
Bus, Train, Bicycle = Transport (horizontal movement)
Elevator = Transport (vertical movement, not road transport)
→ Elevator is odd one out
```

### Type 3 — Grammar Category
```
B, D, F = Consonants
A = Vowel
→ A is odd one out
```

### Type 4 — Mathematical Category
```
216 = 6³ (cube)
343 = 7³ (cube)
512 = 8³ (cube)
625 = 5⁴ (NOT a cube — it's a 4th power)
→ 625 is odd one out
```

### Type 5 — Letter Pattern Category
```
JKLM = 4 consecutive letters
NOPQ = 4 consecutive letters
RSTU = 4 consecutive letters
VWXZ = V,W,X,Z — Y is missing! Not consecutive
→ VWXZ is odd one out
```

### Type 6 — Geography Category
```
Myanmar, Bangladesh, China = Countries bordering India
Thailand = Does NOT border India
→ Thailand is odd one out
```

### Type 7 — Plant Parts vs Types
```
Herb, Tree, Shrub = Types of plants (classification)
Flower = Part of a plant (not a type of plant)
→ Flower is odd one out
```

---

## Step-by-Step Method

```
Step 1: Find the feature that 3 of the 4 options share
Step 2: The one that does NOT share that feature = odd one out
Step 3: If unsure, try multiple features — pick the most specific/exact one
```

## Exam Traps ⚠️
> Don't pick the "most different looking" — find the one that **breaks the rule**
> 625 = 5⁴ — looks like a big cube but is NOT 5³ (5³=125)
> VWXZ — Y is missing — looks consecutive but isn't
> "Flower" in plant question = **part of plant** — Herb/Tree/Shrub are **types** of plants
> Thailand does NOT border India — Myanmar does (easy to confuse)
> Elevator = vertical transport — not same category as road/rail transport
