## ⚡ Flash Rules — Memorize These First

| Rule | Key Point |
|------|-----------|
| Ranking from both ends | Total = Rank from front + Rank from back − 1 |
| Linear seating | Draw a straight line, place people step by step |
| Circular seating | Draw a circle, go clockwise = right |
| Blood relation shortcut | Draw a family tree immediately |
| Direction | Always start facing North unless stated |
| Right turn = clockwise | Left turn = anti-clockwise |
| After 2 right turns | Facing opposite direction |
| Letter series +1 | A→B→C (Z wraps to A) |
| Number series pattern | Check difference, then difference of difference |
| Analogy | Find the exact relationship, then apply to second pair |
| Odd one out | Find ONE feature shared by 3, not 4 |
| Syllogism ALL | If All A=B and All B=C → All A=C |
| Syllogism SOME | Some A=B + All B=C → Some A=C |
| Syllogism NO | If No A=B → No B=A (convertible) |

## 🚨 Common Exam Traps

| Trap | Wrong Thinking | Correct Approach |
|------|---------------|-----------------|
| Ranking total | Add both ranks directly | Rank from front + back − **1** |
| Circular seating right | Think of it as linear | In circle facing centre — right = anti-clockwise direction |
| Blood relation "mother's brother" | Confusion with uncle type | Mother's brother = **Maternal Uncle** |
| Direction after turns | Lose track of turns | Draw compass — mark each turn |
| Series — square numbers | Confuse 1,4,9,16,25 with 2,4,6,8 | Squares: 1²,2²,3²,4²,5² |
| Syllogism "Some A are B" | Conclude "Some B are A" always | **Yes** — Some is always convertible |
| Syllogism "All A are B" | Conclude "All B are A" | **No** — All is NOT reversible |
| Coding +1 shift | Apply same shift to all letters | Check if shift is constant or varies |
| Odd one out | Pick most different looking | Pick the one that breaks the **common rule** |
