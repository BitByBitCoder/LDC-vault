# 10 — Non-Verbal Reasoning
[[🧠 Reasoning — INDEX|← Back to Index]]

> These questions involve figures, shapes, patterns. Cannot be memorized — only method can be learned.

---

## Questions From Your Papers

| Type | What to do |
|------|-----------|
| Figure series — what comes next | Find the pattern of change in each step |
| Odd figure out — which is different | Find feature 3 figures share that 4th doesn't |
| Figure matrix — complete the grid | Find row/column pattern |
| Mirror image | Flip horizontally |
| Paper folding + punch | Unfold mentally step by step |
| Embedded figures | Find given shape hidden in larger figure |

---

## Type 1 — Figure Series

### Method
```
Step 1: Look at what CHANGES from figure 1 to figure 2
Step 2: Check if same change happens from 2 to 3
Step 3: Apply that change to find figure 4 or 5
```

### Common Changes in Figure Series
| Change Type | What to Look For |
|------------|-----------------|
| Rotation | Arrow/shape rotates 45°, 90°, 180° clockwise/anti-clockwise |
| Addition | New element added each step |
| Removal | Element removed each step |
| Size change | Gets bigger or smaller |
| Shading | White→grey→black pattern |
| Direction | Element flips or reverses |

---

## Type 2 — Figure Matrix (3×3 Grid)

### Method
```
Step 1: Look at each ROW — find the pattern across columns
Step 2: Look at each COLUMN — find the pattern down
Step 3: The missing piece must satisfy BOTH row and column patterns
```

### Common Matrix Patterns
| Pattern | Description |
|---------|-------------|
| Same element different sizes | Small, medium, large in each row |
| Rotation | 0°, 90°, 180° rotation across row |
| Addition | 1 element, 2 elements, 3 elements |
| Combination | Row 1 + Row 2 = Row 3 |

## Type 3 — Mirror Image

### Rules
```
Horizontal mirror (left-right flip):
- Left side becomes right side
- Top stays top, bottom stays bottom
- Writing becomes REVERSED horizontally

Vertical mirror (top-bottom flip):
- Top becomes bottom
- Left stays left, right stays right
```

### Common Mistakes
```
❌ Rotating the figure — that's NOT a mirror image
✅ Flipping it horizontally (left becomes right)

If arrow points right → mirror image arrow points left
If arrow points up → mirror image arrow still points up (horizontal flip)
```

## Type 4 — Paper Folding & Punching

### Method
```
Step 1: Start with the FINAL folded state (the punch)
Step 2: UNFOLD one step at a time
Step 3: Each unfold mirrors/reflects the punch marks
Step 4: Count total holes after full unfolding
```

### Folding Rules
```
If paper folded ONCE and punched once → 2 holes when unfolded
If paper folded TWICE and punched once → 4 holes when unfolded
If paper folded THREE TIMES → 8 holes when unfolded
General: n folds → 2ⁿ holes from one punch
```

## Type 5 — Grouping Figures into Classes

### From Your Papers
**Q: Group given 7 figures into 3 classes**

### Method
```
Step 1: Look for ONE feature shared by 2-3 figures
Step 2: Group those together
Step 3: Repeat for remaining figures
Step 4: Check if groups make 3 clean sets
```

### Common Features to Look For
- Same shape (circles, squares, triangles)
- Same shading (filled, unfilled, half-filled)
- Same number of sides
- Same type (geometric, irregular)
- Symbols inside vs outside

## Type 6 — Box/Cube Faces

### From Your Papers
**Q: Which digit appears on face opposite to face showing 4?**

### Method
```
Step 1: Use the unfolded net shown
Step 2: Identify which faces are opposite to each other
Step 3: In a cube: 3 pairs of opposite faces (top-bottom, front-back, left-right)

Standard rule: In an unfolded cross pattern:
- Top and bottom are opposite
- Left and right extremes are opposite
- Middle front and back are opposite
```

## General Strategy for Non-Verbal

```
1. Don't rush — take 30 seconds to study the pattern
2. If figure series: cover answer options, predict the answer, then find it
3. If odd one out: find what 3 have in common first
4. If matrix: check row AND column pattern both
5. If unsure: eliminate clearly wrong answers, then guess from remaining
6. These questions cannot be memorized — practice 10-15 of each type
```

---

## Exam Traps ⚠️
> Mirror image ≠ rotation — they are different transformations
> Paper folded 3 times = **8 holes** from one punch (2³) — not 3 holes
> Figure series — check if rotation is **clockwise or anti-clockwise**
> Matrix — the missing piece must fit **both** the row AND column pattern
> Cube opposite faces — the face shown in unfolded net diagram tells you which faces are adjacent (touching) and which are opposite
