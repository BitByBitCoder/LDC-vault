## Questions From Your Papers

| Analogy | Answer | Relationship |
|---------|--------|-------------|
| Thunder : Rain :: ? : Night | **Dusk** | Thunder precedes Rain; Dusk precedes Night |
| Arc : Circle :: Line : ? | **Rectangle** | Arc is part of Circle; Line is part of Rectangle |
| Flow : River :: Stagnant : ? | **Pool** | Flow is property of River; Stagnant is property of Pool |
| Cup : Lip :: Bird : ? | **Beak** | Cup touches Lip; Bird touches food with Beak |
| Atom : Molecule :: Cell : ? | **Organism** | Atoms make Molecules; Cells make Organisms |
| CUP : LIP :: BIRD : ? | **BEAK** | Tool/part for consuming |

---

## Step-by-Step Method

```
Step 1: Find the EXACT relationship between the first pair
Step 2: State it precisely (not vaguely)
Step 3: Apply the SAME relationship to the second pair
Step 4: Pick the answer that fits
```

## Types of Analogy

### Type 1 — Word Relationships
| Category | Example |
|----------|---------|
| Part to Whole | Arc : Circle :: Line : Rectangle |
| Tool to Purpose | Pen : Write :: Knife : Cut |
| Worker to Tool | Carpenter : Saw :: Painter : Brush |
| Animal to Sound | Dog : Bark :: Cat : Meow |
| Animal to Home | Dog : Kennel :: Bird : Nest |
| Product to Source | Curd : Milk :: Wood : Tree |
| Cause to Effect | Thunder : Rain :: Dusk : Night |
| Singular to Plural | Leaf : Leaves :: Wolf : Wolves |
| Object to Use | Key : Lock :: Switch : Light |
| Word to Meaning | Brave : Courageous :: Sick : Ill |
| Antonym pair | Hot : Cold :: Day : Night |

### Type 2 — Number Relationships
| Example | Relationship |
|---------|-------------|
| 4:16 :: 5:25 | Number : Square |
| 2:8 :: 3:27 | Number : Cube |
| 25:5 :: 64:8 | Number : Square root |
| 11:121 :: 12:144 | Number : Square |
| 3:12 :: 5:20 | ×4 relationship |

### Type 3 — Letter Relationships
| Example | Relationship |
|---------|-------------|
| ACE : BDF :: GIK : HJL | Each letter +1 |
| ABC : ZYX :: DEF : WVU | Mirror/reverse |
| CKO : FNL :: FLP : IOM | +3 to each letter |

## Letter Analogy — Quick Method
```
Find what was done to each letter in first pair:
CKO : FNL
C(3)→F(6) = +3
K(11)→N(14) = +3
O(15)→L(12) = −3 ← different! (or: reverse position from Z)

Check pattern carefully before assuming constant shift
```

---

## Exam Traps ⚠️
> Thunder : Rain ≠ thunder IS rain — it's thunder **precedes** rain (cause/signal relationship)
> Arc : Circle — arc is a **part** of circle, NOT arc is made of circle
> Don't pick the most obvious connection — find the **exact** relationship
> Number analogy — check if it's square, cube, ×n, or something else before answering
> Letter analogy — **count the positions**, don't guess the shift
