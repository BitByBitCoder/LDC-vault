### **Mizoram: A List of "Firsts"**

| **Category**       | **"First" Achievement**       | **Details**                                                                                  |
| ------------------ | ----------------------------- | -------------------------------------------------------------------------------------------- |
| **Literature**     | First Mizo Language Book      | _Mizo Zir Tir Bu_ (Mizo Primer), published on 22 October 1896. Rev. J.H. Lorrain (Pu Buanga) |
| **Literature**     | First Mizo Novel              | _Hawilopari_, written by L. Biakliana in 1936.                                               |
| **Administration** | First Chief Minister          | Ch. Chhunga (took office when Mizoram became a UT in 1972).                                  |
| **Administration** | First Chief Secretary         | R.M. Aggarwal (appointed in 1972).                                                           |
| **Legislature**    | First Assembly Speaker        | H. Thansanga (tenure: 1972–1975).                                                            |
| **Civil Services** | First Mizo IPS Officer        | Rokunga Chawngthu (1962).                                                                    |
| **Civil Services** | First Mizo IAS Officer        | Jamchonga Nampui (cleared the exam in 1954).                                                 |
| **Engineering**    | First Mizo IIT Graduate       | Pu Thanhnuna (graduated from IIT Madras in 1965).                                            |
| **Awards**         | First Mizo Woman (Padma Shri) | Nuchhungi Renthlei                                                                           |
| Awards             | Patma shri                    |  Shri Anselm Sawihlira in 1983.                                                              |
