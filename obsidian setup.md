**Settings → Files and links**

Look for **Default location for new attachments**.

You'll see options like:


- **Same folder as current file** ✅ (Recommended)




### Also make GitHub-friendly links

Still in **Settings → Files and links**:

- **Use [[Wikilinks]]** → **OFF**
- **New link format** → **Relative path to file**

Then pasted images will look like:

```
![](../../Assets/images/Indian%20constitution%20Parts%202.png)
```

which GitHub renders correctly.
