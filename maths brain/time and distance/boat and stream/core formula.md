Here's a focused exam revision on **Boats & Streams** under Time & Distance:

---

## Core Terms

|Term|Meaning|
|---|---|
|**b**|Speed of boat in still water|
|**s**|Speed of stream (current)|
|**D**|Downstream speed = b + s|
|**U**|Upstream speed = b − s|

---

## The 4 Master Formulas

|Find|Formula|
|---|---|
|Downstream speed|D = b + s|
|Upstream speed|U = b − s|
|Boat speed (still water)|b = (D + U) / 2|
|Stream speed|s = (D − U) / 2|

> These four are the backbone of every boats & streams problem.

---

## Time & Distance Application

|Scenario|Formula|
|---|---|
|Time to cover distance `d` downstream|t = d / (b+s)|
|Time to cover distance `d` upstream|t = d / (b−s)|
|Distance covered downstream in time `t`|d = (b+s) × t|
|Round trip time (up + down, same dist `d`)|T = 2bd / (b²−s²)|

---

## Special Cases (Exam Favourites)

**1. Speed ratio given** If downstream : upstream = m : n, then

- Boat speed : Stream speed = (m+n) : (m−n)

**2. Time ratio given** If time upstream : time downstream = m : n (same distance), then

- D : U = n : m (inverse of time ratio)
- b : s = (n+m) : (n−m)

**3. Man rowing in still water** If a man can row `x` km/h in still water and stream is `y` km/h:

- Goes downstream at x+y, upstream at x−y

**4. Boat cannot move upstream when** b = s → upstream speed = 0

---

## Classic Trap Questions

|Trap|Key insight|
|---|---|
|"Average speed for entire journey"|Use 2DU/(D+U), not (D+U)/2 — same distance formula|
|"Stream speed = half boat speed"|s = b/2 → D = 3b/2, U = b/2|
|"Twice as fast downstream as upstream"|D = 2U → b+s = 2(b−s) → s = b/3|
|Time up ≠ time down|Same dist, so t_up > t_down always|

---

## Step-by-Step Method

1. Identify what's given — speeds, time, or distance?
2. If D and U given → use (D+U)/2 and (D−U)/2 to find b and s
3. If b and s given → compute D and U first, then solve
4. For round trip → use T = 2bd/(b²−s²)
5. Always check: upstream speed must be > 0, else boat can't go upstream
