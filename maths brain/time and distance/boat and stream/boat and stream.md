# Boats & Streams

Boats & Streams is a Time & Distance topic where a boat moves in a river with a current (stream).

---

# What is a Stream?

A stream is flowing water.

```text
~~~~~~~~~~~~>
River Current
```

The current can either:

- Help the boat move faster
    
- Slow the boat down
    

---

# Case 1: Downstream

The boat moves **with the current**.

```text
Stream --->

Boat   --->
```

Suppose:

- Boat speed in still water = 20 km/hr
    
- Stream speed = 5 km/hr
    

The current helps the boat.

Downstream Speed:

```text
20 + 5 = 25 km/hr
```

### Rule

```text
Downstream Speed = Boat Speed + Stream Speed
```

---

### Example

Boat speed = 18 km/hr

Stream speed = 4 km/hr

```text
Downstream Speed
= 18 + 4
= 22 km/hr
```

✅ Answer = 22 km/hr

---

# Case 2: Upstream

The boat moves **against the current**.

```text
Boat   --->

<--- Stream
```

The current slows the boat.

Suppose:

- Boat speed = 20 km/hr
    
- Stream speed = 5 km/hr
    

Upstream Speed:

```text
20 - 5 = 15 km/hr
```

### Rule

```text
Upstream Speed = Boat Speed - Stream Speed
```

---

### Example

Boat speed = 18 km/hr

Stream speed = 4 km/hr

```text
Upstream Speed
= 18 - 4
= 14 km/hr
```

✅ Answer = 14 km/hr

---

# Finding Boat Speed

Suppose:

- Downstream Speed = 30 km/hr
    
- Upstream Speed = 20 km/hr
    

Add them:

```text
30 + 20 = 50
```

Divide by 2:

```text
50 / 2 = 25
```

### Rule

```text
Boat Speed
= (Downstream Speed + Upstream Speed) / 2
```

✅ Boat Speed = 25 km/hr

---

# Finding Stream Speed

Suppose:

- Downstream Speed = 30 km/hr
    
- Upstream Speed = 20 km/hr
    

Subtract them:

```text
30 - 20 = 10
```

Divide by 2:

```text
10 / 2 = 5
```

### Rule

```text
Stream Speed
= (Downstream Speed - Upstream Speed) / 2
```

✅ Stream Speed = 5 km/hr

---

# Example

Downstream Speed = 18 km/hr

Upstream Speed = 10 km/hr

Boat Speed:

```text
(18 + 10) / 2
= 28 / 2
= 14 km/hr
```

Stream Speed:

```text
(18 - 10) / 2
= 8 / 2
= 4 km/hr
```

✅ Boat Speed = 14 km/hr

✅ Stream Speed = 4 km/hr

# Mini Test

### 1.

Boat Speed = 12 km/hr

Stream Speed = 3 km/hr

Find downstream speed.

### 2.

Boat Speed = 15 km/hr

Stream Speed = 4 km/hr

Find upstream speed.

### 3.

Downstream Speed = 24 km/hr

Upstream Speed = 16 km/hr

Find:

- Boat Speed
    
- Stream Speed
    

---

# Answers

### 1

```text
12 + 3 = 15 km/hr
```

✅ 15 km/hr

### 2

```text
15 - 4 = 11 km/hr
```

✅ 11 km/hr

### 3

Boat Speed:

```text
(24 + 16) / 2
= 40 / 2
= 20 km/hr
```

Stream Speed:

```text
(24 - 16) / 2
= 8 / 2
= 4 km/hr
```

✅ Boat Speed = 20 km/hr

✅ Stream Speed = 4 km/hr

---

### Formulas Summary

```text
Downstream Speed = Boat Speed + Stream Speed

Upstream Speed = Boat Speed - Stream Speed

Boat Speed = (Downstream Speed + Upstream Speed) / 2

Stream Speed = (Downstream Speed - Upstream Speed) / 2

Time = Distance / Speed

Distance = Speed × Time
```
