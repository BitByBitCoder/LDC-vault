# What is Relative Speed?

Relative speed means:

> "How fast one object is moving with respect to another object."

Instead of looking at one train, we look at **how quickly the distance between two objects changes**.

---

# Case 1: Moving Towards Each Other

Suppose:

- Train A = 60 km/hr
    
- Train B = 40 km/hr
    

They move towards each other.

```
A ---> <--- B
60       40
```

Every hour:

- A covers 60 km
    
- B covers 40 km
    

Distance between them decreases by:

[  
60 + 40 = 100  
]

Relative Speed:

[  
100 \text{ km/hr}  
]

### Rule

[  
\text{Relative Speed} = \text{Sum of Speeds}  
]

when moving **towards each other**.

---

### Example

Distance between two trains = 500 km

Speeds:

- 60 km/hr
    
- 40 km/hr
    

Relative speed:

[  
60+40=100  
]

Time to meet:

[  
\frac{500}{100}=5  
]

✅ Answer = 5 hours

---

# Case 2: Moving In Same Direction

Suppose:

```
A -----> 80 km/hr

B -----> 50 km/hr
```

Both move in the same direction.

Every hour:

- A gains 80 km
    
- B gains 50 km
    

A gains on B by:

[  
80-50=30  
]

Relative Speed:

[  
30 \text{ km/hr}  
]

### Rule

[  
\text{Relative Speed} = \text{Difference of Speeds}  
]

when moving **in the same direction**.

---

### Example

A is 150 km behind B.

Speeds:

- A = 80 km/hr
    
- B = 50 km/hr
    

Relative speed:

[  
80-50=30  
]

Time to catch B:

[  
\frac{150}{30}=5  
]

✅ Answer = 5 hours

---

# Train Example

Train A = 80 km/hr

Train B = 100 km/hr

Moving towards each other.

Relative speed:

[  
80+100=180  
]

If distance is 540 km:

[  
\frac{540}{180}=3  
]

✅ They meet in 3 hours.

# Mini Test

Try these yourself:

1. Two cars are 300 km apart and move towards each other at 80 km/hr and 70 km/hr. How long until they meet?
    
2. A bus travels at 60 km/hr and a car at 90 km/hr in the same direction. The car is 120 km behind the bus. How long to catch it?
    

Don't look below until you've tried.

---

### Answers

[  
80+70=150  
]

[  
300/150=2  
]

✅ 2 hours

[                          
90-60=30  
]

[  
120/30=4  
]

✅ 4 hours
