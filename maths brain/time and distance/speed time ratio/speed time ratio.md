# Speed-Time Ratio

This is one of the easiest Time & Distance concepts.

It is based on one formula:

```text
Distance = Speed × Time
```

---

# Key Idea

When the **distance is fixed**:

```text
Speed ∝ 1 / Time
```

This means:

> If speed increases, time decreases.

> If speed decreases, time increases.

So Speed and Time are **inversely proportional**.

---

# Rule

If:

```text
Speed Ratio = a : b
```

Then:

```text
Time Ratio = b : a
```

Just **reverse the ratio**.

---

# Example 1

Two cars travel the same distance.

Their speeds are:

```text
60 : 40
```

Time ratio:

```text
40 : 60
```

Simplify:

```text
2 : 3
```

✅ Answer = 2 : 3

---

# Example 2

A and B cover the same distance.

Speed ratio:

```text
3 : 5
```

Time ratio:

```text
5 : 3
```

✅ Answer = 5 : 3

---

# Easy Memory Trick

```text
Speed ↑  → Time ↓

Speed ↓  → Time ↑
```

So:

```text
Speed Ratio = 3 : 4

Time Ratio = 4 : 3
```

Just flip the ratio.

---

# Finding Time Saved

Suppose a man increases his speed from:

```text
40 km/hr to 50 km/hr
```

Speed ratio:

```text
40 : 50

= 4 : 5
```

Time ratio:

```text
5 : 4
```

This means:

```text
Old Time : New Time = 5 : 4
```

Time saved:

```text
5 - 4 = 1 part
```

---

# Shortcut Formula

If speed changes from:

```text
a → b
```

Then:

```text
Time Saved Percentage
= ((b - a) / b) × 100
```

---

# Example

Speed increases:

```text
20 km/hr → 25 km/hr
```

Time saved:

```text
((25 - 20) / 25) × 100

= (5 / 25) × 100

= 20%
```

✅ Time decreases by 20%

### Golden Rule

```text
For the same distance:

Speed Ratio = a : b

Time Ratio = b : a
```

**Just reverse the ratio.** This single idea solves most Speed-Time Ratio questions. 🚀
