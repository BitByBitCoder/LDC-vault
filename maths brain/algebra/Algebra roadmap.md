# PHASE 4 — Factorization

## Goal

Break expressions into factors.

### Topics

- Common factor method
    
- Grouping
    
- Using identities
    
- Quadratic factorization
    

### Examples

[  
x^2+5x+6  
]

becomes:

[  
(x+2)(x+3)  
]

Very important for exams.
