# Most Important Topics for Exams

If you're preparing for SSC, Banking, Railway, LDC, State Exams, focus on:

```text
1. Linear Equations
2. Algebraic Identities
3. Factorization
4. Quadratic Equations
5. Indices (Powers)
6. Ratio through Algebra
7. Age Problems
8. Word Problems
```

---

# 1. Simple Linear Equations

Find the value of x.

### Example

```text
2x + 5 = 15
```

```text
2x = 10

x = 5
```

✅ Very common in exams.

---

# 2. Algebraic Identities

Important formulas to memorize:

```text
(a + b)² = a² + 2ab + b²

(a - b)² = a² - 2ab + b²

(a + b)(a - b) = a² - b²

(a + b)³ = a³ + 3a²b + 3ab² + b³

(a - b)³ = a³ - 3a²b + 3ab² - b³
```

### Example

```text
99²

= (100 - 1)²

= 10000 - 200 + 1

= 9801
```

---

# 3. Factorization

Breaking an expression into factors.

### Example

```text
x² - 9

= x² - 3²

= (x + 3)(x - 3)
```

---

# 4. Quadratic Equations

Equations of the form:

```text
ax² + bx + c = 0
```

### Example

```text
x² - 5x + 6 = 0

(x - 2)(x - 3) = 0

x = 2 or 3
```

---

# 5. Simplification of Expressions

### Example

```text
(2x + 3x) - x

= 5x - x

= 4x
```

---

# 6. Surds and Indices

### Indices

```text
a^m × a^n = a^(m+n)

a^m / a^n = a^(m-n)

(a^m)^n = a^(mn)
```

### Example

```text
2³ × 2²

= 2⁵

= 32
```

# 8. Ratio Using Algebra

### Example

Two numbers are in ratio 3:5.

Let numbers be:

```text
3x and 5x
```

If their sum is 40:

```text
3x + 5x = 40

8x = 40

x = 5
```

Numbers:

```text
15 and 25
```

# 9. Age Problems

### Example

A father is 3 times as old as his son.

Let son's age be:

```text
x
```

Father's age:

```text
3x
```

If total age is 48:

```text
x + 3x = 48

4x = 48

x = 12
```

Son = 12

Father = 36

---

# 10. Word Problems

Convert English into equations.

### Example

A number increased by 7 is 20.

```text
x + 7 = 20

x = 13
```
