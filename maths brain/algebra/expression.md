# Simplifying Expressions

Simplifying means:

- combine similar terms
    
- remove unnecessary parts
    
- make the expression shorter and cleaner
    

---

# Step 1 — Understand Terms

Expression:

[  
2x + 3x  
]

Here:

- (2x) is a term
    
- (3x) is another term
    

Since both have (x), they are **like terms**.

---

# Rule of Like Terms

You can only combine terms with the SAME variable part.

Examples:

|Can Combine|Result|
|---|---|
|(2x + 3x)|(5x)|
|(4y - y)|(3y)|
|(5a + 2a)|(7a)|

---

# Cannot Combine

[  
2x + 3y  
]

Different variables.

So it stays:

[  
2x + 3y  
]

---

# Examples

## Example 1

2x+5x

Add coefficients:

[  
2+5=7  
]

Answer:

[  
7x  
]

---

## Example 2

7a-3a

[  
7-3=4  
]

Answer:

[  
4a  
]

---

## Example 3

3x+2y+x

Combine only (x) terms:

[  
3x+x=4x  
]

Answer:

[  
4x+2y  
]

---

# Step 2 — Constants

Numbers without variables are constants.

Example:

x+2+3

[  
2+3=5  
]

Answer:

[  
x+5  
]

---

# Step 3 — Combine Everything

Example:

2x+3+4x-1

Combine (x) terms:

[  
2x+4x=6x  
]

Combine constants:

[  
3-1=2  
]

Answer:

[  
6x+2  
]

---

# Step 4 — Using Brackets

Example:

2(x+3)

Multiply 2 into bracket:

[  
2\times x + 2\times 3  
]

Answer:

[  
2x+6  
]

This is called **distribution**.

---

# Negative Sign Rule

Example:

-(x+4)

Negative changes signs inside:

[  
-x-4  
]

---

# Bigger Example

2(x+3)+x

First expand:

[  
2x+6+x  
]

Combine like terms:

[  
3x+6  
]

---

# Common Mistake

Wrong:

[  
2x+3=x5  
]

You cannot combine variable term with constant.

Correct:

[  
2x+3  
]
