# Linear Equation

A **linear equation** is an equation where the highest power of the variable is **1**.

### Examples

```text
x + 5 = 12

2x - 3 = 7

5x + 4 = 19
```

These are linear equations because **x has power 1**.

---

# Why is it called "Linear"?

Because when graphed, it forms a **straight line**.

Examples:

```text
y = x

y = 2x + 3

y = 5x - 1
```

But for competitive exams, you usually just solve for x.

---

# Example 1

```text
x + 5 = 12
```

Subtract 5 from both sides:

```text
x = 12 - 5

x = 7
```

✅ Answer = 7

---

# Example 2

```text
3x = 21
```

Divide both sides by 3:

```text
x = 21 / 3

x = 7
```

✅ Answer = 7

---

# Example 3

```text
2x + 3 = 11
```

Subtract 3:

```text
2x = 8
```

Divide by 2:

```text
x = 4
```

✅ Answer = 4

---

# Not Linear Equations

These are **not** linear because the power of x is greater than 1:

```text
x² + 5 = 10

x³ - 2 = 0

√x + 1 = 5
```

These belong to other topics like quadratic equations.

---

# Standard Form

A linear equation is often written as:

```text
ax + b = 0
```

where:

- a ≠ 0
    
- a and b are numbers
    

### Example

```text
3x - 9 = 0
```

Solve:

```text
3x = 9

x = 3
```

---

# Quick Exam Trick

For a linear equation:

```text
ax + b = c
```

Use:

```text
x = (c - b) / a
```

### Example

```text
4x + 6 = 22
```

```text
x = (22 - 6) / 4

x = 16 / 4

x = 4
```

✅ Answer = 4

### Golden Rule

```text
Linear Equation = Variable has power 1
```

If you see only **x** (not x², x³, √x, etc.), it's usually a linear equation.
