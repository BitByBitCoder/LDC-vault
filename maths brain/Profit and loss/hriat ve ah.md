This is an excellent LDC question because it uses the **1% method**.

Let's solve it slowly.

---

# Step 1: Read carefully

> The difference in selling prices when sold at:
> 
> - **15% profit**
>     
> - **17% loss**
>     
> 
> is **₹96**.

Many students think ₹96 is the selling price.

❌ It is **not**.

It is the **difference between two selling prices**.

---

# Step 2: Draw the percentages

Both selling prices are based on the **same Cost Price**.

```text
Cost Price = 100%

Selling Price 1 = 115%   (15% profit)

Selling Price 2 = 83%    (17% loss)
```

---

# Step 3: Find the percentage difference

Draw it.

```text
115%
│
│  ← Difference
│
83%
```

Difference

[  
115%-83%=32%  
]

So the question is actually saying

```text
32% = ₹96
```

⭐ This is the most important step.

---

# Step 4: Find 1%

Since

```text
32% = ₹96
```

then

[  
1%=\frac{96}{32}=3  
]

```text
1% = ₹3
```

---

# Step 5: Find 110%

The question asks for **10% profit**.

That means

```text
CP = 100%

SP = 110%
```

Now

```text
110% = 110 × ₹3
```

[  
=330  
]

---

# ✅ Final Answer

[  
\boxed{₹330}  
]

---

# Recognition Trick

Whenever a question says

> Difference between two profits/losses

Don't think about money first.

Think about **percentage difference**.

Example:

|Situation|Percentage|
|---|--:|
|15% profit|115%|
|17% loss|83%|
|Difference|**32%**|

Then write

```text
32% = Given Difference
```

Find **1%**, and then find whatever percentage the question asks for.

---

## ⭐ This is one of the most common LDC/SSC patterns

1. Convert profit/loss into **selling-price percentages**.
    
2. Find the **percentage difference**.
    
3. Write:
    

```text
Percentage Difference = Given ₹ Difference
```

4. Find **1%**.
    
5. Find the required percentage (100%, 110%, 120%, etc.).
    

This method is much faster than finding the Cost Price first.
