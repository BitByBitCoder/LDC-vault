## Profit & Loss — All Formulas for LDC

---

### FORMULA 1 — Basic Profit & Loss

```
Profit = SP - CP
Loss   = CP - SP

Profit% = (Profit / CP) × 100
Loss%   = (Loss / CP)   × 100
```

**Golden Rule: % is ALWAYS calculated on CP, never SP**

Q1, Q2 directly test this.

---
	
### FORMULA 2 — Finding CP from SP and Profit%/Loss%

```
CP = (SP × 100) / (100 + Profit%)   ← when profit given
CP = (SP × 100) / (100 - Loss%)     ← when loss given
```

Q3, Q4, Q5, Q6 all use this exact formula. This is the single most repeated formula in your papers.

---

### FORMULA 3 — Finding SP from CP and Profit%/Loss%

```
SP = CP × (100 + Profit%) / 100    ← when profit given
SP = CP × (100 - Loss%)   / 100    ← when loss given
```

---

### FORMULA 4 — Ratio of CP to SP

```
If Profit% = P
CP : SP = 100 : (100 + P)

If Loss% = L
CP : SP = 100 : (100 - L)
```

Q7, Q8, Q10, Q11 test this directly.

---

### FORMULA 5 — When SP is Equal for Two Items (Most Dangerous Trap)

```
When two items sold at SAME SP
one at X% profit, other at X% loss

ALWAYS a NET LOSS
Loss% = X² / 100
```

Example: Both sold at same price, one +20%, other -20% Loss% = 20²/100 = **4% loss always**

Q41, Q42, Q43, Q44, Q49 — all same formula. **This trap appears in every single Mizoram LDC paper.**

---

### FORMULA 6 — SP of Articles Formula (High Frequency)

```
If SP of X articles = CP of Y articles
Profit% = (Y - X) / X × 100   ← if Y > X = profit
Loss%   = (X - Y) / Y × 100   ← if Y < X = loss  (note: denominator changes!)

wait — cleaner version:
Profit% = [(Y/X) - 1] × 100
Loss%   = [1 - (Y/X)] × 100
```

Q23, Q24 directly test this.

---

### FORMULA 7 — Metres/Articles Gain Formula

```
By selling X metres, gain = CP of Y metres
Gain% = Y / (X - Y) × 100

By selling X metres, gain = SP of Y metres
Gain% = Y / (X - Y) × 100  ← same formula, different trap
```

Q25, Q26, Q27, Q28 test this.

---

### FORMULA 8 — Overhead/Repair Costs (Add to CP)

```
Actual CP = Purchase Price + Repair + Transport + any other cost
Then apply normal profit/loss formula on this total CP
```

Q16, Q17 test this.

---

### FORMULA 9 — Difference in SP Formula

```
If selling at A% profit and B% loss gives difference of Rs.D

CP = D × 100 / (A + B)

Then find SP at required %
```

Q20, Q21 test this.

---

### FORMULA 10 — Dishonest Dealer (False Weight)

```
Gain% = (True Weight - False Weight) / False Weight × 100
```

Appeared in your uploaded papers (Legal Metrology paper had this directly).

---

### FORMULA 11 — Discount Formula

```
Discount = Marked Price - Selling Price
Discount% = (Discount / Marked Price) × 100

SP = MP × (100 - Discount%) / 100
MP = SP × 100 / (100 - Discount%)
```

---

### FORMULA 12 — Successive Discounts

```
Two discounts A% and B%:
Net = A + B - (AB/100)

Three discounts: apply twice
```
