Yes! The divisibility rule for 7 is a bit trickier than most, but here it is:

## The Rule

**Double the last digit, subtract it from the rest of the number. If the result is divisible by 7 (including 0), the original number is too.**

Repeat the process if the number is still too large to judge.

---

### Example: Is 343 divisible by 7?

1. Last digit: **3** → double it = **6**
2. Remaining number: **34** → subtract: 34 − 6 = **28**
3. 28 ÷ 7 = 4 ✅ → Yes, 343 is divisible by 7.

---

### Example: Is 623 divisible by 7?

1. Last digit: **3** → double = **6**
2. 62 − 6 = **56**
3. 56 ÷ 7 = 8 ✅ → Yes.

---

### Example: Is 500 divisible by 7?

1. Last digit: **0** → double = **0**
2. 50 − 0 = **50**
3. Last digit of 50: **0** → double = **0** → 5 − 0 = **5**
4. 5 is not divisible by 7 ❌ → No.

---

	**Tip:** Unlike rules for 2, 3, 5, or 9, the rule for 7 requires a few steps — so for small numbers it's often just as fast to directly divide. This rule shines for larger numbers.
