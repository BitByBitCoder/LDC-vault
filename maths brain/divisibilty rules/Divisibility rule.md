## Divisibility Rules — Sub-topics

**1. Basic Rules (2, 3, 5, 10)** Simple single-condition checks. Most common in exams.

**2. Composite Divisibility (36, 12, 24, 72...)** Break into co-prime factors. Like 36 = 4 × 9. The problem you just solved falls here.

**3. Alternating Sum Rule (11)** Add/subtract digits alternately.

**4. Iterative/Recursive Rules (7, 13, 17, 19)** Requires repeated steps — double last digit, subtract, repeat.

**5. Sum of Digits Rules (3, 9)** Just add all digits and check.

**6. Last Digit Rules (2, 5, 10)** Only the last digit matters.

**7. Last Two Digits Rules (4, 25)** Only last two digits matter.

**8. Last Three Digits Rules (8, 125)** Only last three digits matter.

**9. Finding Unknown Digits** Like your question — given a number with x and y, find values using divisibility conditions. Very common in LDC/SSC exams.

**10. Divisibility by Powers (4, 8, 16 / 9, 27)** Based on powers of 2 or 3.

---

**For LDC**, the highest frequency topics are:

- Rules for 2, 3, 4, 5, 8, 9, 11
- Finding unknown digits (like today's problem)
- Composite divisibility (36, 12, 24)
