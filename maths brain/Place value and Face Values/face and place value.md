### Face Value

The **face value** of a digit is the digit itself.

Example:

```text
5487
```

- Face value of 5 = 5
    
- Face value of 4 = 4
    
- Face value of 8 = 8
    
- Face value of 7 = 7
    

✅ Face value never changes.

---

### Place Value

The **place value** depends on the position of the digit.

Example:

```text
5487
```

|Digit|Place|Place Value|
|---|---|---|
|5|Thousands|5000|
|4|Hundreds|400|
|8|Tens|80|
|7|Ones|7|

✅ Place value changes with position.

---

### Example

In **72543**:

- Face value of 2 = **2**
    
- Place value of 2 = **2000**
    

because 2 is in the thousands place.

---

### Quick Rule

```text
Face Value = Digit itself

Place Value = Digit × Value of its position
```

Example:

```text
6432
```

- Face value of 4 = 4
    
- Place value of 4 = 400
    

	This is a very common basic aptitude question in LDC exams.
