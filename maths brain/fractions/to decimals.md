To convert a **fraction to a decimal**, simply **divide the numerator by the denominator**.

[  
\text{Decimal} = \frac{\text{Numerator}}{\text{Denominator}}  
]

### Examples

**1.** ( \frac{1}{2} )

[  
1 \div 2 = 0.5  
]

Answer: **0.5**

---

**2.** ( \frac{3}{4} )

[  
3 \div 4 = 0.75  
]

Answer: **0.75**

---

**3.** ( \frac{1}{5} )

[  
1 \div 5 = 0.2  
]

Answer: **0.2**

---

**4.** ( \frac{3}{8} )

[  
3 \div 8 = 0.375  
]

Answer: **0.375**

---

### Relationship

Fraction → Decimal → Percentage

[  
\frac{3}{4} = 0.75 = 75%  
]

[  
\frac{1}{5} = 0.2 = 20%  
]
