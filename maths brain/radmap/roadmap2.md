## 4. Exact Roadmap — No Fixed Date, Built on Volume Not Calendar

Since you have no fixed exam date, this is structured in **work blocks**, not weeks. Each block = until you hit 85% accuracy on timed drills, then move on — don't linger.

**Block 1 (Computer Knowledge — biggest section, attack first): Hardware + OS** Memorize CPU/storage/device facts, Windows file management, shortcuts. This is pure recall — fastest marks available in the whole exam.

**Block 2: MS Word + MS Excel** Formatting, AutoCorrect, Mail Merge, formulas (SUM/COUNT/AVERAGE), cell referencing, charts. These two alone = 30.8% of Computer Knowledge.

**Block 3: PowerPoint + Internet/Web + Email** Slide views, layouts; browser/search engine basics; email components. Quick, finite content.

**Block 4: Arithmetic — Percentage + Profit/Loss/Discount** These two repeat constantly with the same patterns (successive discounts, marked price, gain%, salary comparison traps). Drill until automatic.

**Block 5: Average + Ratio/Proportion + Number System** Replacement-average problems, ratio chains, LCM/HCF/divisibility, digit problems.

**Block 6: Speed/Distance/Time + Time & Work** Train crossing, boats/streams, pipes/cisterns, work-together problems.

**Block 7: SI/CI + Ages + Geometry basics** Just the formulas and 2-3 question types each, not deep theory.

**Block 8: Reasoning — Seating Arrangement + Blood Relations + Series** This is the highest-yield reasoning block. Practice circular and linear arrangements heavily.

**Block 9: Direction Sense + Classification + Coding-Decoding + Non-Verbal** Finite question types, fast to learn once seen a few times.

**Block 10: Syllogism + remaining low-frequency types** Last, lowest ROI, accept partial mastery here.

**Final Block: Full timed mocks only** Take complete papers from this question bank under exam conditions, review every wrong answer, repeat until consistent 80%+.
