## PERCENTAGE — Complete Formula Sheet for LDC

---

### FORMULA 1 — Basic Percentage

```
% of a number = (% × Number) / 100
```

**Covers Q2, Q4, Q12**

---

### FORMULA 2 — Percentage of a Fraction

```
What % is A of B = (A/B) × 100
```

**Covers Q1, Q6, Q9, Q10**

Q1 trick: convert everything to same unit first 8 hours of 3 days → 3 days = 72 hours → (8/72) × 100 = 11.1%

---

### FORMULA 3 — Finding Original Number

```
If X% of number = given value
Then number = (given value × 100) / X
```

**Covers Q2, Q3, Q5, Q11, Q12, Q13, Q14**

Q3 specific pattern: 75% of N + 75 = N → 75 = N - 75%N = 25%N → N = 300

---

### FORMULA 4 — Percentage Increase/Decrease

```
% Increase = (Increase / Original) × 100
% Decrease = (Decrease / Original) × 100
```

**Covers Q15, Q22, Q25, Q26**

---

### FORMULA 5 — New Value Given % Increase

```
New Value = Original × (100 + %) / 100
Original = New Value × 100 / (100 + %)
```

**Covers Q24, Q27, Q28, Q31**

Q24/Q27: Previous salary = 55000 × 100/110 = 50000

---

### FORMULA 6 — New Value Given % Decrease

```
New Value = Original × (100 − %) / 100
Original = New Value × 100 / (100 − %)
```

**Covers Q13, Q29, Q30, Q32**

Q30: Original = 440 × 100/80 = 550

---

### FORMULA 7 — Salary Trap Formula (THE MOST COMMON LDC TRAP)

```
If A is X% less than B
Then B is more than A by = X/(100−X) × 100 %
```

Example: A is 20% less than B → B is more than A by 20/80 × 100 = **25%**

This appears in almost every paper. Memorize this cold.

---

### FORMULA 8 — Successive % Change (Net Effect)

```
Net effect = a + b + (ab/100) %
```

Where a and b are the two % changes (use − for decrease)

**Covers Q41, Q42, Q43, Q44, Q45**

Q42: −10 then +10 = −10 + 10 + (−10×10)/100 = **−1%** (net decrease) Q44: +20 then −25 = 20 + (−25) + (20×−25)/100 = −5 + (−5) = **−10%**

For three successive discounts (Q45): Apply formula twice — first two together, then result with third.

---

### FORMULA 9 — Spending + Saving Type

```
Income = Expenditure + Savings
If savings = X% of income → Expenditure = (100−X)% of income
```

**Covers Q16, Q35, Q36**

Q16: Saves 12.5% → spends 87.5% → 87.5% of income = 3500 → income = 4000

---

### FORMULA 10 — Population/Pass-Fail Type

```
Total who failed = 100% − (weighted average of those who passed)
```

**Covers Q17, Q34**
