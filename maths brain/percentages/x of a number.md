This is another:

# "x% of a number + constant = number"

question.

Let the number be (N).

Given:

[  
75% \text{ of } N + 75 = N  
]

Convert 75% to a fraction:

[  
\frac{75}{100}N + 75 = N  
]

[  
\frac{3}{4}N + 75 = N  
]

Move (\frac{3}{4}N) to the right:

[  
75 = N - \frac{3}{4}N  
]

[  
75 = \frac{1}{4}N  
]

Multiply both sides by 4:

[  
N = 75 \times 4  
]

[  
N = 300  
]

✅ **Answer: 300**

### Shortcut

If:

[  
75% \text{ of a number} + 75 = \text{the number}  
]

then the remaining part is:

[  
100% - 75% = 25%  
]

So:

[  
25% = 75  
]

[  
100% = 75 \times 4 = 300  
]

This shortcut is much faster in exams.
