The question:

> **35% of a number is 112. Find the number.**

belongs to the percentage type:

# Finding the Whole When a Percentage Is Given

or simply:

# "x% of a number = given value"

### Standard Form

[  
x% \text{ of a number } = \text{given value}  
]

Example:

[  
35% \text{ of a number } = 112  
]

Let the number be (N).

[  
\frac{35}{100}\times N = 112  
]

[  
N = 112 \times \frac{100}{35}  
]

[  
N = 320  
]

✅ **Answer: 320**

Your question is:

> **"x% of a number = value"**

This is one of the most common LDC percentage question types. A quick shortcut is:

[  
\text{Number}=\frac{\text{Given Value}\times100}{\text{Percentage}}  
]

So:

[  
\frac{112\times100}{35}=320  
]

No equation needed once you get comfortable with the formula.
