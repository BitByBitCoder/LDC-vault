For LDC and most competitive exams, **Simplification** usually means solving calculations quickly and accurately.

## Topics You Should Study

### 1. BODMAS / Order of Operations

Learn the correct order:

**B**rackets → **O**f (powers) → **D**ivision → **M**ultiplication → **A**ddition → **S**ubtraction

Example:

```
8 + 12 ÷ 3 × 2

= 8 + 4 × 2
= 8 + 8
= 16
```

---

### 2. Fractions

Addition, subtraction, multiplication, division.

Examples:

```
3/4 + 5/4 = 8/4 = 2

2/3 × 3/5 = 6/15 = 2/5

4/5 ÷ 2/3
= 4/5 × 3/2
= 12/10
= 6/5
```

---

### 3. Decimals

Examples:

```
2.5 + 1.75 = 4.25

3.6 × 2 = 7.2

4.8 ÷ 0.2 = 24
```

---

### 4. Percentages

Examples:

```
25% of 200 = 50

40% of 150 = 60

75% = 3/4
```

---

### 5. Squares

Memorize:

```
11² = 121
12² = 144
13² = 169
...
30² = 900
```

At minimum, learn squares up to 30.

---

### 6. Square Roots

Examples:

```
√144 = 12
√225 = 15
√625 = 25
```

Memorize square roots of common perfect squares.

---

### 7. Cubes

Examples:

```
2³ = 8
3³ = 27
4³ = 64
...
20³ = 8000
```

---

### 8. Cube Roots

Examples:

```
∛125 = 5
∛216 = 6
∛729 = 9
```

---

### 9. Algebraic Simplification

Basic expressions:

```
2x + 3x = 5x

7a - 2a = 5a

3(x + 2)
= 3x + 6
```

## What to Prioritize for LDC

Study in this order:

1. BODMAS
    
2. Fractions
    
3. Decimals
    
4. Percentages
    
5. Squares (1–30)
    
6. Square Roots
    
7. Cubes (1–20)
    
8. Cube Roots
    
9. Basic Algebra
    

If you're just starting, focus on **BODMAS + Fractions + Percentages + Squares/Square Roots** first. Those topics usually give the highest return in simplification questions.
