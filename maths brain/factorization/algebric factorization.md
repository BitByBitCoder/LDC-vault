# Factorization

Factorization means:

> Breaking a big expression into smaller multiplying parts.

It is the reverse of expansion.

---

# Example Idea

Expansion:

[  
(x+2)(x+3)  
]

becomes:

[  
x^2+5x+6  
]

Factorization does the opposite:

[  
x^2+5x+6  
]

becomes:

[  
(x+2)(x+3)  
]

---

# Why Factorization Is Important

Used in:

- solving equations
    
- simplifying algebra
    
- quadratic equations
    
- competitive exams
    

---

# Method 1 — Common Factor

Take out the common part.

Example:

2x+4

Both terms contain 2.

Take 2 outside:

[  
2(x+2)  
]

---

# Another Example

5x+10

Common factor = 5

Answer:

[  
5(x+2)  
]

---

# Method 3 — Quadratic Factorization

Example:

x^2+5x+6

We need two numbers:

- whose multiplication = 6
    
- whose addition = 5
    

Those numbers are:

- 2
    
- 3
    

So:

[  
(x+2)(x+3)  
]

---

# Another Example

x^2+7x+12

Need:

- multiplication = 12
    
- addition = 7
    

Numbers:

- 3
    
- 4
    

Answer:

[  
(x+3)(x+4)  
]

---

# Negative Example

x^2-x-6

Need:

- multiplication = -6
    
- addition = -1
    

Numbers:

- -3
    
- 2
    

Answer:

[  
(x-3)(x+2)  
]

---

# How to Check

Multiply back.

Example:

[  
(x+2)(x+3)  
]

Multiply:

[  
x^2+3x+2x+6  
]

[  
x^2+5x+6  
]

Correct.
